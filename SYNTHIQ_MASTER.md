## 2. THE FULL PIPELINE

```
FIND → RESEARCH → BUILD → PITCH → TRACK → CLOSE
```

**Step 1 — FIND**
Use the find_businesses skill to search Google Maps for local businesses with no website.
Filter: no website, within 25 miles of Augusta, GA.

**Step 2 — RESEARCH**
Run research_business skill on each lead.
Pulls: real photos, verbatim Google reviews, logo, brand colors, hours, phone, address, owner name, taglines, services.

**Step 3 — BUILD**
Generate a unique demo site per business using the category-aware design system.
- 14 business categories each with unique color palette, font, hero layout
- 10 hero layout types — never repeat
- Real photos, real reviews, clickable phone, mobile-first

**Step 4 — PITCH**
Send cold email from Synthiq101@gmail.com using one of 4 templates (see below).
Email contains direct link to their personalized demo site.

**Step 5 — TRACK**
Dashboard monitors every lead:
- found → site_built → approved → contacted → replied → converted
- Gmail reply watcher automation fires when they reply back

**Step 6 — CLOSE**
Follow up, customize, negotiate, go live. Est. $1,200/client average.

## 5. LEAD DATABASE SCHEMA

Every BusinessLead record contains:

| Field | Type | Description |
|---|---|---|
| business_name | string | Business name |
| owner_name | string | Owner name if found |
| phone | string | Click-to-call phone number |
| email | string | Contact email for outreach |
| address | string | Full street address |
| city | string | City (Augusta, Evans, Martinez, Grovetown) |
| category | string | Business type |
| google_rating | number | Google star rating |
| review_count | number | Total Google reviews |
| photo_urls | array | Real photos from Google Maps |
| google_reviews | array | Review text (verbatim) |
| google_reviews_full | array | Full review objects with author + rating |
| demo_site_slug | string | URL slug for their demo site |
| demo_site_url | string | Full demo site URL |
| email_sent | boolean | Whether pitch was sent |
| email_sent_date | string | Date pitch was sent |
| email_replied | boolean | Whether they replied |
| email_reply_text | string | Their reply text |
| email_reply_date | string | Date they replied |
| status | enum | found → site_built → approved → contacted → replied → converted → skipped |
| approved | boolean | Manual approval flag |
| notes | string | Internal notes |
